Twitch Chat Downloader

Enter the video ID in URL to download the chat in JSON format

ex) 123456789 in "twitch.tv/videos/123456789"

Note 1: Because of pagination, only 50~60 chats can be downloaded with one API call. For videos with a large number of chats, it may take a few minutes to download all chats.

Note 2: This page uses Twitch's unofficial chat API endpoint
